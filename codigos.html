Códigos de radio y procedimientos operativos estándar

📻🔹 𝐂𝐎́𝐃𝐈𝐆𝐎𝐒 𝐃𝐄 𝐑𝐀𝐃𝐈𝐎 - 𝐂𝐎́𝐃𝐈𝐆𝐎𝐒 𝐑𝐀𝐃𝐈𝐀𝐋𝐄𝐍 🔹📻

🔹 Códigos Básicos

🔹 Código 1️⃣: Acudir o confirmar el último aviso. (Respuesta vía radio)

🔹 Código 2️⃣: Acudir a un aviso de rutina, sin luces ni sirenas.

🔹 Código 3️⃣: Acudir a un aviso de rutina, con luces (rotativos).

🔹 Código 4️⃣: Acudir a una emergencia, con luces y sirenas.

🔹 Código 5️⃣: No solicitamos más apoyo en la zona.

🔹 Código 6️⃣: Alejarse de la zona de operación.

🔹 Código 7️⃣: Realizamos una pausa (descanso).

🔹 Código 9️⃣: Sujeto en búsqueda y captura.

🔹 Código 9️⃣ Adam: Sujeto en búsqueda y captura peligroso.

🔹 Código 🔟: Silencio absoluto en la radio.

🔹 Código 1️⃣2️⃣: Falsa alarma.

🔹 Código 3️⃣0️⃣: Enfrentamiento armado (tiroteo).

🔹 Código 3️⃣7️⃣: Vehículo reportado como robado.

🔹 Código 1️⃣0️⃣0️⃣: Posible bloqueo/intercepción de vehículo en persecución.

🔹 Código 1️⃣8️⃣7️⃣: Homicidio o intento de homicidio.

🔹 Código 2️⃣0️⃣7️⃣: Secuestro en curso.

🔹 Código 2️⃣1️⃣1️⃣: Robo a establecimiento o banco.

🔹 Código 2️⃣1️⃣5️⃣: Agresión a una persona.

🔹 Código 2️⃣4️⃣0️⃣: Asalto con arma blanca.

🔹 Código 2️⃣4️⃣5️⃣: Asalto con arma de fuego.

🔹 Código 2️⃣4️⃣7️⃣: Disparos detectados.

🔹 Código 2️⃣5️⃣4️⃣ Víctor: Persecución con vehículo.

🔹 Código 2️⃣5️⃣4️⃣ Paul: Persecución a pie.

🔹 Código 3️⃣2️⃣0️⃣: Venta de sustancias ilegales.

🔹 Código 4️⃣5️⃣9️⃣: Allanamiento de morada.

🔹 Código 4️⃣8️⃣7️⃣: Robo al banco federal.

🔹 Código 4️⃣8️⃣8️⃣: Hurto o robo menor.

🔹 Código 5️⃣0️⃣2️⃣: Conducción bajo los efectos de sustancias (alcohol, drogas, medicación…)
